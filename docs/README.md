# G-Helper-HimpqEnhanced

一个G-Helper日常使用的增强版本。

## Features
 - 增加启动GHelper后默认的性能模式设置，解决启动后静音模式让电脑初启动时卡顿的问题。
 - 增加断电切换性能模式的开关
 - 增加可自选的电源管理计划调整UI，此功能在GHelper中以手动修改配置文件的形式实现
 - 添加任务栏性能监控/浮窗性能监控，类似于TrafficMonitor的功能
 - 修改赞助按钮为HE版本的设置按钮
